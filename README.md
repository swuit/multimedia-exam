# 多媒体技术应用教程考试重点

本书为西南大学多媒体技术应用教程考试重点，方便同学们复习。

## 考试题型

- 选择
- 填空
- 判断
- 综合

## 考试范围

- 全书1-10章
- 不考具体操作部分

## 完善本书

本书所有章节已经基本写完了，但是写的过程中难免有各种错误与不足，希望大家在看的时候可以顺便指出。
